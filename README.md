# Что это
Песенник для детского лагеря. Сверстан с помощью latex.

# Структура

- `book.tex` основная книга
- `changelog.tex` страница с последними изменениями
- `gtrcrd.sty` `guitarchordschemes.sty` `guitartabs.cls` `gtrcrd-doc.tex` библиотеки для аккордов, схем, табулатур. Отличаются от оригинальных, переписаны под песенник
- `testN.txt` тестовые книги, котоыре я частенько использую чтобы сверстать одну страницу, проверить отрисовку табулатур или проверить оглавление. По хорошему не должны быть в репозитории, но мне лень выделять под это отдельную папку и настраивать компиляцию
- `out/` директория куда складывается скомпилированные книги и промежуточные файлы
- `img/` `other/` директории с картинками

# Сборка
## Overleaf
Проект вёлся в https://www.overleaf.com, но сайт сильно порезал время компиляции.  
Вся книга уже не влезает в бесплатный тариф.  
Если оплатить (~1200 рублей), всё быстро скомпилируется в браузере.

## Локальная сборка
Требуется: 
- tex live
- visual studio code
- latex workhop (расширение в visual studio code)

В расширении будет кнопка `build LaTeX project`.  
**Сборка может закончиться ошибкой**, но book.pdf всё равно появитсья в `out/`

note: Могут работать и другие инструменты, проверен только этот набор.  
**Warning!**: Установка tex live занимает вечность. Я установил пакеты ~12 часов

## версия для просмтора и для печати
Версия для печати требует чтобы первая страница была справа. Однако большинство приложений с двухстраничным просмотром этого не учитывают. Поэтому в `book.tex` Находится липовая нулевая страница.  
Чтобы сдулать версию для печати, нужно либо  
 - а) убрать страницу в любом редакторе pdf  
 - б) убрать в `book.tex` строчки начиная с `\setcounter{page}{0}` и заканчивая `\newpage`, пересобрать книгу

## Ошибки
При сборке могут быть ошибки и предупреждения. Тем не менее, pdf всё равно сгенерируется.
- `Command \sh already defined.` - Какой-то конфликт пакетов. Так и не смог понять что с чем пересекается. Времени потрачено: 5ч
- `Underfull \hbox` - незаполненное горизонтальное пространства. Такие предупреждения успешно игнорируются по всей книге. Причина: слабые знания latex автора песенника
- `Underfull \vbox` - незаполненное вертикальное пространство. Same

# Как это кодить
Текст пишеся как есть на естественном языке и дополняется latex командами начинающимися с `\`. Все команды приринмают в себя параметры в фигурных скобка `{}`. Для каких-то это специальные значения, а для каких-то просто текст с которым они работают. Существуют ещё области `\begin{}`.. `\end{}` которые позволяют задать целые области со своими правилами.
## Общие latex команды/символы
- `\\` - перенос строки
- `\textbf{blablabla}` - жирный текст
- `\newpage` - новая страница
- `--` Тире
- ` ``blablabla''` кавычки
## Оглавление
- `\addcontentsline{toc}{section}{Раздел}` - добавление раздела в оглавление
- `\addcontentsline{toc}{subsection}{Песня - Автор}` - добавление песни в оглавление
- `\addtocontents{toc}{\protect\newpage}` - добавление разрыва строки в оглавление. Если оглавление не влезает в страницу, оно может не сгенерироваться вообще.
## Оступы
- `\\` - Вертикальный стандартные оступ, просто пустая строка. 
- `\musVspace`- Вертикальный **уменьшенный** отступ. Ставлю между куплетами/припевами если приходится ужать текст песни. Эквивалентен `\vspace{2mm}`
- `\vspace{-5mm}` Вертикальный **отрицательный** отступ. Ставлю после заголовка если приходится очень сильно ужать текст песни. Заставляет всю песню поднятся повыше и прилипнуть к заголовку.
- `\-\hspace{10mm}` - Горизонтальный отступ для припевов. TODO: надо бы придумать короткую команду
- `\-\hspace{3mm}` - Горизонтальный отступ для соседних куплетов.  TODO: надо бы придумать короткую команду
## Музыкальное 
### Аккорды
- `Am{}` `Dm{}` `C{}`... - Отображенеи классических аккордов. Они будут над текстом в фигурных скобках
- `\CHORD{Am7}{}` `\CHORD{Am G}{}` - отображение неклассических аккордов (или нескольких рядом). Они будут над текстом во вторых фигурных скобках
### Табы
Проще скопировать примеры из книги чем описать
- `\begin {tabline}{n}{}{}{E,A,D,G,H,E}`... `\end {tabline}` Начать область табулатур из n струн, подписанных E A D G H E
- `\note{i}{s}{n}{x}` - поставить `i`-ую по порядку "ноту", отступив `1/s`-ую часть блока от предыдущей, на `n` струне и подписать ее текстом `x`
- `\nextbar` разделитель блоков
### } 2 раза
Чтобы получить фигурную скобку и подпись "2 раза", приходится входить в математический режим latex. Это неудобно и всё постоянно хочет съехать, поэтому костыли:
- Вся песня заключается в `\begin{gather*}` ... `\end{gather*}` чтобы математический режим не добавлял нумерацию
- Повторяющйися кусок песни заключается в `\begin{musRepeat}` ... `\end{musRepeat}` что рисует математическу фигурную скобку и подпись "2 раза"
- Каждая строчка заключается в `\textup{blablabla}`, что заставляет отрисовать текст в математическом режиме. Так мы избегаем непотяных отступов между стандартным текстом и математическим (костыль, но никто не замечает). Внутри textup могут работать все остальные команды.
