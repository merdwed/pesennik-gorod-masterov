# Структура

- `book.tex` основная книга
- `changelog.tex` страница с последними изменениями
- `gtrcrd.sty` `guitarchordschemes.sty` `guitartabs.cls` `gtrcrd-doc.tex` библиотеки для аккордов, схем, табулатур. Отличаются от оригинальных, переписаны под песенник
- `testN.txt` тестовые книги, котоыре я частенько использую чтобы сверстать одну страницу, проверить отрисовку табулатур или проверить оглавление. По хорошему не должны быть в репозитории, но мне лень выделять под это отдельную папку и настраивать компиляцию
- `out/` директория куда складывается скомпилированные книги и промежуточные файлы
- `img/` `other/` директории с картинками


# Локальная сборка
Потребуется: 
- tex live
- visual studio code
- latex workhop (расширение в visual studio code)

В расширении будет кнопка `build LaTeX project`. 
**Сборка может закончиться ошибкой**, но book.pdf всё равно появитсья в `out/`

note: Могут работать и другие инструменты, проверен только этот набор. Установка tex live занимает вечность.

## версия для просмтора и для печати
Версия для печати требует чтобы первая страница была справа. Однако большинство приложений с двухстраничным просмотром этого не учитывают. Поэтому в `bool.tex` Находится липовая первая страница, обычно заполненная changelog
Чтобы получить версию для печати, нужно либо 
- убрать страницу в любом редакторе pdf
- убрать всю страницу начиная с `\setcounter{page}{0} % начать нумерацию с номера 0` и заканчивая `\newpage`, скомпилировать книгу
